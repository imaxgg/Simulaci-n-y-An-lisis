# Simulación y análisis de señales con Transformada de Fourier

Proyecto que analiza señales en dominio del tiempo y frecuencia usando Python.

Señales simuladas:
- Pulso rectangular
- Escalón unitario
- Señal senoidal

Se aplicó FFT para obtener:
- Magnitud
- Fase

Propiedades verificadas:
- Linealidad
- Desplazamiento temporal
- Escalamiento en frecuencia
